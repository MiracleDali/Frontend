Date对象

Date: JavaScript 原生时间库

  • getTime(): 返回实例距离 1970年1月1日00:00:00 的毫秒数
  • getDate(): 返回实例对象对应每个月的几号（从1开始）
  • getDay(): 返回星期几，星期日为0，星期一为1，以此类推
  • getYear(): 返回距离1900的年数
  • getFullYear(): 返回四位的年份
  • getMonth(): 返回月份（0表示1月，11表示12月）
  • getHours(): 返回小时（0-23）
  • getMilliseconds(): 返回毫秒（0-999）
  • getMinutes(): 返回分钟（0-59）
  • getSeconds(): 返回秒（0-59）
