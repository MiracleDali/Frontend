ES6 简介与 JavaScript 的关系

JavaScript 与 ECMAScript 的关系

  • JavaScript 是一种广泛使用的编程语言，主要用于Web开发
  • ECMAScript 是 JavaScript 的标准化规范，由 ECMA 国际组织制定
  • JavaScript 是 ECMAScript 规范的具体实现之一

ES6 (ECMAScript 2015) 概述

ES6（也称为 ECMAScript 2015 或 ES2015）是 JavaScript 语言的一个重要版本，于2015年发布

ES6 由 TC39 委员会制定，该委员会负责 ECMAScript 标准的演进。https://github.com/tc39 包含了标准提案、会议记录以及规范的最新进展。

主要特性包括：

  1. let 和 const 命令
  2. 变量的解构赋值
  3. 字符串扩展
  4. 函数扩展
  5. 对象扩展
  6. 数组扩展
  7. 运算符扩展
  8. Promise对象
  9. Class
  10. Class 继承
  11. ...
