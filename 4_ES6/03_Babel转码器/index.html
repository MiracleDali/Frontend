Babel转码器

Babel 是一个 JavaScript 编译器，主要用于将 ES6+ 语法转换为向后兼容的 JavaScript 代码，使现代 JavaScript 代码能够在旧版浏览器或环境中正常运行。



  • 安装Babel在项目的根目录下
    npm install --save-dev @babel/core @babel/cli
  • 配置文件 .babelrc
  • 转码规则-presets字段转码规则, 官方提供以下的规则集，根据需要安装
    npm install --save-dev @babel/preset-env
  • 将规则加入到 .babelrc
    { "presets": ["@babel/env"], "plugins": [] }

安装 Babel 命令行转码

npm install --save-dev @babel/cli
